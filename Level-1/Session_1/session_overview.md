## 📚 Theory Part

**Focus:** Conceptual understanding of backend development

### Topics Covered:

#### 1. Introduction & Resources
- Recap of previous session (OSC, Backend Committee)
- Tour of resources (WhatsApp, GitHub, Discord, Google Sheets, Forms)
- Committee rules and reward system

#### 2. What is Backend?
- Definition and core concepts
- Backend vs Frontend differences
- Client-server communication
- Other communication patterns (server-to-server, async messaging, real-time)
- Backend responsibilities: Receive → Process → Store → Respond

#### 3. Backend Engineer Mindset
- How backend engineers think
- Core responsibilities (API design, databases, security, performance)
- Required attitude and skills
- Problem-solving approach

#### 4. Languages & Tools Overview
- **Languages covered:**
  - JavaScript/Node.js
  - Python (Django, Flask)
  - Java (Spring Boot)
  - C#/.NET
  - Go (Golang)
  - PHP (Laravel, WordPress)
  - Ruby (Ruby on Rails)

#### 5. Essential Terminology
- **Language:** Syntax and rules for writing code
- **Runtime Environment:** Where code actually executes (Browser, Node.js, etc.)
- **Framework:** Pre-built structure for applications
- **Ecosystem:** Tools, resources, and community around a language

#### 6. JavaScript Deep Dive
- What is JavaScript?
- Where JavaScript runs:
  - **Browser Runtime:** JS Engine, Web APIs, DOM manipulation
  - **Node.js Runtime:** V8 Engine, Node APIs, server-side capabilities
- How browsers execute JavaScript
- Why JavaScript is sent as text (not machine code)

## 💻 Technical Part


**Focus:** Practical JavaScript programming skills

### Topics Covered:

#### 1. JavaScript & Node.js Basics
- Brief intro to JavaScript
- Key characteristics of JavaScript
- Backend with Node.js
- Setting up and running code

#### 2. Core JavaScript Concepts

##### Variables & Data Types
- Values & Variables (`let`, `const`, `var`)
- Primitive types (Number, String, Boolean, Undefined, Null, BigInt, Symbol)
- Non-primitive types (Objects, Arrays)
- Type checking with `typeof`

##### Operators & Logic
- Basic operators (arithmetic, assignment, comparison, logical)
- Operator precedence
- If/else statements
- Boolean logic & equality operators (`==` vs `===`)

##### Type System
- Type conversion (explicit)
- Type coercion (implicit)
- Statements vs Expressions

#### 3. Working with Strings
- Basic string methods:
  - Character access: `charAt()`, `charCodeAt()`
  - String operations: `concat()`, `includes()`, `indexOf()`, `lastIndexOf()`
  - Case conversion: `toUpperCase()`, `toLowerCase()`
  - Extraction: `substring()`, `slice()`, `split()`
  - Cleaning: `trim()`, `trimStart()`, `trimEnd()`
  - Padding: `padStart()`, `padEnd()`
  - Replacing: `replace()`, `replaceAll()`
- Template literals for string interpolation

#### 4. Functions
- Regular functions
- Arrow functions
- Differences between function types
- `this` binding
- Implicit returns

#### 5. Data Structures
- Introduction to Arrays
- Introduction to Objects
- Basic array operations

#### 6. Control Flow
- For loops
- While loops
- Loop control and iteration
