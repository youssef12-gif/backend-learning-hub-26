## What is BackEnd-Learning-Hub ?

This Repo is your guide to become a backend developer. Designed for our OSC Committee members who want to learn backend track, takes you on a juorny starting  from Backend basics all the way to advanced backend topics & projects.

## 🎯 What You'll Learn

- **Level 1**: Core backend fundamentals such as TypeScript ,databases, and authentication.
- **Level 2**: Advanced backend topics such as including NestJS, webSockets, and Docker.

## 🚀 Getting Started

### Before starting, make sure you have:

- **Node.js** (v18+ recommended) - [Download](https://nodejs.org/)
- **Git** - [Download](https://git-scm.com/)
- **Code Editor** - [VS Code](https://code.visualstudio.com/) recommended

### Your Learning Journey 

1. **Start with Level 1, Session 01** - Don't skip sessions!
2. **Read Theory First** - Understand concepts before coding
3. **Follow Code Examples** - Type them out, don't copy-paste
4. **Complete Exercises** - Practice makes perfect
5. **Build Projects** - Apply what you've learned
6. **Review Solutions** - Learn from different approaches

## 🏅 How to Join the Challenge

1. **View Badges & Their Descriptions**

2. **Choose a Badge Challenge**

3. **Upload Your Code**

4. **Submit Your Work**
