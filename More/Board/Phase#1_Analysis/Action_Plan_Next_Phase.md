## Planned Actions

### 1. session preparation
* start preb at least one week before session 
* all session materils(slides , docs, tasks, hands on) must be done two days before the session
* at least one rehersal before the session
* all sessions must follow the acceptance criteria
    
### 2. Tasks
* no tasks will be generated using AI
* task must take maximum 2 hours estimated to solve it
* the solution to the task must be ready before announcing the task
* all Tasks must follow the acceptance criteria (Ai section)
* task feedback must be published max 1 day before next session

### 3. Hands on
* no Hands on will be generated using AI
* Hands on must take maximum 10 mins estimated to solve it
* the solution to the Hands on must be ready before the session max two days
* all Hands on must follow the acceptance criteria (Ai section)

### 4. content documantation 
* no document will be generated using AI
* documnet should be reviwed by at least two board members 
* all documents must follow the acceptance criteria (Ai section)

### 5. engagment
